# N14 · 트렌딩 키워드 카드 {#card-n14}

> 지금 사람들이 많이 보는 키워드 묶음. 인기 순위와 별개로 변동성·신규 진입 키워드 중심.

## 목적

내 관심사 너머 **세상의 흐름**을 작은 신호로 보여주기. 모든 사용자에게 동일하게 노출되는 거시적 시그널.

## 변종

- **all** — 전체 사용자 기준 트렌딩 키워드
- **adjacent** — 내 구독 키워드와 인접한 영역의 트렌딩만

## 노출 요소

- 키워드 칩 ×N (선택 가능)
- 각 칩 옆 meta: "사용자 12.3k명" 또는 "+340%"
- 칩 탭 시 채널뷰 진입 + 임시 구독 옵션 노출

## 데이터

```ts
TrendingData: { scope: 'all' | 'adjacent', items: [{ keyword, meta }] }
```

## 노출 규칙

- 새소식 피드 하루 1회 정도가 적정. 너무 자주 노출하면 거시 피드처럼 보임
