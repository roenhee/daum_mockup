# N18 · 속보 카드 {#card-n18}

> 사용자 관심 키워드에 속보 발생 시 우선 노출되는 짧은 카드.

## 목적

데일리 브리핑(N17)과 달리 **즉시성**이 핵심. 발생 직후 짧은 요약 + 빠른 재생.

## 노출 요소

- 빨간 그라데이션 배경 ("속보" 시각 코드)
- 단일 키워드 칩
- 요약 1~2줄
- 짧은 재생 버튼 (보통 30초~1분)
- 푸터: 기반 매체 수

## 데이터

```ts
FlashData: { duration, keyword, summary, basedOnCount }
```

## 노출 규칙

- 알림으로도 푸시 (별도 정책 — `policy.md` 의 알림 권한 섹션 참조)
- 피드 최상단 강제 노출. 사용자가 dismiss 하기 전까지는 sticky
