# N4 · 키워드 추천 카드 {#card-n4}

> 사용자의 행동을 기반으로 새 구독 키워드를 제안하는 카드.

## 목적

사용자의 관심 영역을 자연스럽게 확장. 단순 "추천"이 아니라 **추천 이유**를 함께 노출해서 수긍 가능한 제안으로 만든다.

## 변종

- **behavior** — 최근 사용자 행동(저장·반응 등)에서 추출한 키워드
- **adjacent** — 기존 구독 키워드와 인접한 영역에서 확장

## 노출 요소

- 상단 `observation` — "왜 이 추천이 나왔는지" 한 줄 (예: "최근 저장한 기사 8건 중 5건이 호르무즈 관련")
- 키워드 칩 ×N (`{ keyword, description }`)
- 각 키워드 우측 `구독` 버튼

## 데이터

```ts
KeywordRecData: { variant, observation, keywords: [{ keyword, description }] }
```

## 상호작용

- 키워드 칩 탭 → 해당 키워드 채널뷰
- `구독` 탭 → 즉시 구독 + 카드 시각적 dismiss 처리(미구현)
