# M:AI 탭 정책

## 데이터 수집 동의 {#mai-policy-data}

M:AI는 사용자의 다음 데이터를 활용해 개인화 추천을 제공한다.

- 최근 본 기사 (`history.recent`)
- 좋아요·저장한 기사 (`history.reaction`, `history.saved`)
- 구독 채널 목록 (`subscribe.channels`)
- 댓글 활동 (`history.comments`)

수집 동의 거부 시 → 비개인화 인기 피드로 폴백. 거부 상태는 설정 > 데이터 관리에서 확인 가능.

## 알림 권한 {#mai-policy-notification}

운영체제 알림 권한이 필요. 권한 없을 시 알림 아이콘 우상단에 ⚠ 인디케이터 표시.

- 새 글 알림: 구독 채널의 새 글 (기본 ON)
- 이슈 알림: 등록한 이슈에 새 기사 추가될 때 (기본 OFF)
- AI 탐구 완료 알림: 사용자가 요청한 탐구 작업 완료 시 (기본 ON)

## 데이터 보관 {#mai-policy-retention}

- 히스토리: 90일 보관 후 자동 삭제
- 탐구 결과: 사용자가 명시적으로 아카이브하지 않으면 30일 후 삭제
- 사용자가 탈퇴 요청 시 즉시 모든 데이터 익명화 처리
